# 古典密码  

>Enigma  
先转动齿轮再对键盘输入信号进行加密  
接线板的连接只影响键盘输入以及灯泡输出

齿轮内部的设置为：ring setting  
外部：message key  
delta = message key - ring setting  
信号进入齿轮要加上delta , 信号出来时减去delta

1.单个字节不区分大小端。

2. 宽度达到2个字节以上的变量做赋值、与、异或、非等不改变位的位置的操作时，运算结果在大小端机器中无区别。
